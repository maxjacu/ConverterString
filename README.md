# ConverterString
Converts multiple measurements using the one unit supplied at the end of the string 
e.g. ' 20 x 34 x 10 cm' to '7.9 x 13.4 x 3.9 in'

  - Currently supports following
    - m to ft
    - cm to inch
    - cm to inch
    - ml to floz
    
### Install

run as shiny app, requires packages are: 
shiny, birk, stringr
